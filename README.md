# Charlie's AI Agent with Memory

## Setup Requirements
- n8n instance
- MemMachine running on port 8080
- OpenAI API key

## Installation
1. Import the workflow JSON into n8n
2. Update the following:
   - API endpoints if not using localhost
   - Session IDs and user IDs
   - OpenAI credentials

## Workflow Components
- Episodic memory search and storage
- Profile memory support
- Conversation history formatting
